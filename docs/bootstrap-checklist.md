# Bootstrap Checklist

## Shared First Hour

1. Agree on frontend and backend package managers and scripts.
2. Lock `article-draft` and `validation-result` shapes.
3. Add two local scenario folders under `data/scenarios`.
4. Confirm which fields are extracted by AI in the first milestone.
5. Confirm the 7 blocking quality rules.
6. Assign an integration owner.

## Milestone 1

1. Frontend can load a stubbed article draft.
2. Backend can return scenario metadata.
3. Shared types compile in both frontend and backend.
4. Validation can run against a stubbed article draft.

## Milestone 2

1. Backend returns extracted article draft from mocked scenario input.
2. Frontend renders draft and blocking issues.
3. Supplier corrections can be submitted for revalidation.

## Milestone 3

1. Scenario A reaches `Ready for internal approval`.
2. Scenario B demonstrates multiple blocking issues before correction.
3. Minimal internal status view works.
